Step 3 — Add to Gmail

After clicking Copy Signature above:
  1. Open Gmail and click the gear icon (top right) → See all settings
  2. Go to the General tab and scroll down to Signature
  3. Click Create new and name it "Mizzentop"
  4. Click inside the signature box and paste (Ctrl+V on Windows, Cmd+V on Mac)
  5. Under Signature defaults, set it as default for New emails only — leave replies set to none
  6. Scroll to the bottom and click Save Changes
